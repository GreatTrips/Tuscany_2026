# Tuscany 2026 — Trip Itinerary

Group of 10 · April 11–19, 2026 · Florence · Palaia · Chianti · Siena

## How to publish on GitHub Pages

1. Create a new GitHub repository (e.g. `tuscany-2026`)
2. Upload `index.html` to the repository root
3. Go to **Settings → Pages**
4. Under **Source**, select **Deploy from a branch**
5. Choose **main** branch, **/ (root)** folder → click **Save**
6. Your site will be live at: `https://YOUR-USERNAME.github.io/tuscany-2026/`

Share the URL with the group — no login required to view.

## To update content

Edit `index.html` directly on GitHub (click the file → pencil icon) and commit changes. The site updates automatically within ~60 seconds.
